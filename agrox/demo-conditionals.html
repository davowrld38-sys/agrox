Conditional Logic Demo: If/Else Statements in Form Validation

This demo shows how conditional logic (if/else statements) controls form behavior and validation.

function validateStep(stepNumber) { let isValid = true; const errors = []; // IF the current step is 1 (Basic Info) if (stepNumber === 1) { const title = document.getElementById('listing-title').value.trim(); const category = document.getElementById('listing-category').value; const quantity = document.getElementById('listing-quantity').value; const unit = document.getElementById('listing-unit').value; // IF title is empty OR null if (!title) { errors.push('Product title is required'); isValid = false; // Set validation to false } // IF category is not selected if (!category) { errors.push('Category is required'); isValid = false; } // IF quantity is invalid (empty, zero, or negative) if (!quantity || quantity <= 0) { errors.push('Valid quantity is required'); isValid = false; } // IF unit is not selected if (!unit) { errors.push('Unit is required'); isValid = false; } } // ELSE IF the current step is 2 (Pricing) else if (stepNumber === 2) { const price = document.getElementById('listing-price').value; const priceUnit = document.getElementById('price-unit').value; // IF price is invalid if (!price || price <= 0) { errors.push('Valid price is required'); isValid = false; } // IF price unit is not selected if (!priceUnit) { errors.push('Price unit is required'); isValid = false; } } // ELSE IF the current step is 3 (Details) else if (stepNumber === 3) { const description = document.getElementById('listing-description').value.trim(); const location = document.getElementById('listing-location').value.trim(); // IF description is empty if (!description) { errors.push('Product description is required'); isValid = false; } // IF location is empty if (!location) { errors.push('Location is required'); isValid = false; } } // ELSE (step 4 - Review step) else { // Review step validation (could add more logic here) } // IF there are validation errors if (!isValid) { showValidationErrors(errors); // Display errors to user } else { hideValidationErrors(); // Clear any existing errors } return isValid; // Return true if valid, false if invalid }

Conditional Logic Flow

✅ IF Condition Met (Valid)
• Form data passes all checks
• isValid remains true
• User can proceed to next step
• No error messages shown
• Progress indicator updates
❌ ELSE Condition (Invalid)
• Form data fails validation
• isValid set to false
• User blocked from proceeding
• Error messages displayed
• Form highlights problematic fields
